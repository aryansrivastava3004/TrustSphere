# TrustSphere

A trust-based collaboration platform designed to help students find reliable collaborators, build credibility, and work together with transparency.

---

## Problem

Students often struggle to find trustworthy collaborators for projects.
Existing systems lack accountability, verified feedback, and structured dispute resolution, leading to poor collaboration and unreliable outcomes.

---

## Solution

TrustSphere introduces a structured system where users can:

* Create profiles and showcase skills
* Find and connect with collaborators
* Send and manage collaboration requests
* Rate and review each other after task completion
* Build a dynamic trust score based on activity and feedback
* Resolve disputes through a structured system

---

## Key Features

* Trust Score System based on ratings and performance
* Two-way Review System for accountability
* Smart Search & Filtering for collaborator discovery
* Request & Interaction Flow for structured communication
* Dispute Resolution System for conflict handling
